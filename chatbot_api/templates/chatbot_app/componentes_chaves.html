Estrutura do Projeto

Este projeto é organizado por pastas: "App" e "chatbot_api" são as seções principais, cada uma com sua função.

Arquivos como manage.py são ferramentas de controle, e o db.sqlite3 armazena os dados.

Essa organização simplifica o desenvolvimento e a manutenção do projeto.

Esta estrutura é do framework Django.
